Javascript 中右移0位可以用来快速去掉小数，关于位移运算的定义：

> 右移运算是将一个二进制位的操作数按指定移动的位数向右移动，移出位被丢弃，左边移出的空位或者一律补0，或者补符号位。

实际看下效果：

```` javascript
32.5 >> 0  // 32

````

## 右移是什么作用？

```` javascript
(32).toString('2') // 转换为2进制，得到 100000

( 32 >> 1 ).toString(2) // 右移1位，得到 10000

````

对比下：

```` javascript
100000  // 实际值 32
010000  // 实际值 16

````

怎么去掉小数位？

    32.5 >>1 << 1

或者直接

    32.5 >> 0

## 其他的方法？

    ~~ 32.5
